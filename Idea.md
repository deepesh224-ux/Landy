## Project: Landy (Small Landlord & Tenant Portal)
# Core Concept
# A full-cycle "PropTech" platform that handles property discovery (Marketplace) and ongoing management (Maintenance/Payments) using a unique Property Code system for security and onboarding.
## üì± 1. The Tenant Journey
* Screen A: The Public Marketplace
* Feature: Browse available listings.
* Options: Filter by price, location, and amenities.
* Action: "Apply Now" button to send a digital profile to the landlord.
* Screen B: The Onboarding (Your Image)
* Feature: "Join Your Home" portal.
* Action: Tenant enters the 6-digit Property Code generated by the landlord.
* Benefit: Instantly links the tenant to the specific property data, chat, and lease.
* Screen C: The Tenant Dashboard
* Maintenance: Submit "Ticket" with photo, description, and urgency (Low, Normal, Emergency).
* Finances: View rent due dates and a "Pay Now" button (Stripe).
* Chat: Private messaging thread with the landlord.
* Documents: View digital lease and move-in inspection photos.
